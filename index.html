== Infix to postfix conversion (step by step) ==

Directions

This app can create postfix notation of an expression. For example, expression a+(b*c) is abc*+ in postfix notation. You will see an animation step by step.

You can change the infix notation and see the result. See also Expression Tree Creation related to this.
